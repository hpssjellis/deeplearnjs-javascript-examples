K Nearest Neighbor (KNN) in pure JavaScript

From http://burakkanber.com/blog/machine-learning-in-js-k-nearest-neighbor-part-1/

Program takes a few seconds to load. It loads a pre-defined dataset of homes with the following key and then sorts unknown properties into color categories based on the K Nearest Neighbor Algorithm.

Color Key:
  • Apartments are red
  • Houses are green
  • Flats are blue

    Note: Rooms along x axis
    Area along y axis
